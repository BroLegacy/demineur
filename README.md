# Démineur

Un projet de fin d'année en groupe 

## Déployer le projet

Cloner le projet

```bash
  git clone https://github.com/BroLegacy/demineur.git
```

Allez dans le dossier du projet

```bash
  cd demineur
```

Lancer le projet

```bash
  python3 test.py
```
